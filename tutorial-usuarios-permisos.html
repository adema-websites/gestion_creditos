Alta de Usuarios y Permisos

Configura usuarios adicionales y gestiona permisos de acceso para mantener la seguridad de tu sistema.

Video Tutorial

Video Tutorial - Usuarios y Permisos

Pasos Detallados

Sigue estos pasos para gestionar usuarios y permisos de manera segura

1

Crear Nuevo Usuario

Accede a la sección de administración de usuarios y crea un nuevo usuario ingresando nombre de usuario, contraseña y datos básicos del empleado.

Seguridad:

Utiliza contraseñas seguras y únicas para cada usuario. El sistema te guiará sobre los requisitos mínimos de seguridad.

2

Asignar Permisos y Roles

Define qué secciones del sistema puede acceder cada usuario. Puedes crear roles personalizados o usar los predefinidos: Administrador, Operador, Solo Lectura.

Roles disponibles:

Administrador (acceso total), Operador (gestión de clientes y créditos), Solo Lectura (consulta de información).

3

Probar y Validar Accesos

Una vez configurado el usuario, realiza pruebas de acceso para verificar que los permisos funcionen correctamente y el usuario pueda trabajar sin inconvenientes.

Verificación:

Prueba el acceso con las credenciales del nuevo usuario para confirmar que solo puede acceder a las secciones autorizadas.
